Getting started

How do Comets work?

Need to get started quickly with Comets? You will learn all the basics in just minutes.

Arduino Nano

Every Comet Kit comes with an Arduino Nano. The Nano is a small, complete, and breadboard-friendly board based on the ATmega328P. It has more or less the same functionality of the Arduino Duemilanove, but in a different package. It lacks only a DC power jack, and works with a Mini-B USB cable instead of a standard one.


Powering your Comet

The Comet Kit comes with an external 5V (2.1A) 6000maH power supply. This is to ensure that the Comet has enough power to run all of its components. The power supply is connected to the Arduino Nano via a USB cable.


The Comet board

The Comet board is a custom PCB designed to have the Arduino Nano plugged into it through all its GPIO Pins. The board not only comes with all the electrical components, it also acts as a built-in modular breadboard, so you can easily connect your own components without needing to solder anything.


Comet app

The Comet app is a electron.js interface that allows you to control your Comet straight from your computer. The app is built using the latest web technologies and is designed to be easy to use with the ability to perform complex actions.
